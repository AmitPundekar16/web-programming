Section 2: Dynamically Populating a Website from an Array

Rendering Arrays as HTML

APIs usually return arrays of items. To display them on a webpage, we transform each item into an HTML string using .map(), combine them with .join(''), and insert them via innerHTML.

// Pattern: Array of URLs → HTML image cards
const urls = ["https://dog.ceo/.../img1.jpg", "https://dog.ceo/.../img2.jpg"];

const html = urls.map(url => `
    <div class="dog-card">
        <img src="${url}" alt="dog">
    </div>
`).join('');

document.getElementById('output').innerHTML = html;

With a larger API response, data.message is an array of URLs — you pick how many to display and what info to show per card.

.map() + .join('') is the standard pattern for rendering lists. .map() transforms each array item into an HTML string. .join('') glues them into one big string. Then innerHTML inserts it into the page.

Summary of Concepts

Concept What You Learned
onclick event handler Links a button to a function: element.onclick = myFunction;. The function runs every time the button is clicked.
fetch() Makes HTTP requests to APIs. Returns a Promise. Use response.json() to parse the body.
async/await Write asynchronous code that looks synchronous. await pauses until a Promise resolves.
try...catch Error handling for async code. try wraps risky code, catch handles failures.
response.ok Check if the HTTP status is 2xx before using the data. Always throw on !response.ok to trigger your catch block.
URL Parameters Insert variables into URLs using template literals: `https://api.example.com/breed/${breed}/list`
Multiple fetch calls Fetch two endpoints independently in the same function, then combine the results before rendering.
.map() + .join('') Transforms each array item into HTML. .join('') combines into one string for innerHTML.
Dynamic dropdowns Populate a <select> element with <option> tags built from API data; read the chosen value with .value.
Chained API calls Use the result of one interaction (user picks from dropdown) to build the URL for the next fetch call.
Template Literals `Hello ${name}` — embed expressions in strings. Essential for building dynamic HTML and API URLs.

API Used in This Lab

  • Dog CEO Dog API — Free, open REST API for random dog images organised by breed (no authentication required)

Key Endpoints

Endpoint Returns
/api/breeds/image/random One random dog image URL (string)
/api/breeds/image/random/{count} Array of N random image URLs
/api/breed/{breed}/images/random One random image URL for a specific breed
/api/breed/{breed}/list Array of sub-breed names for the breed
/api/breed/{breed}/{subBreed}/images/random/{count} Array of N images for a specific sub-breed
/api/breeds/list/all Object mapping every breed to its sub-breeds array
